git init - создать проект репозитория (создается единожды для проекта)
git status - команда, чтобы понимать текущее состояние git
git add file - команда, чтобы file стал отслеживаемым
git commit - закоммитить отслеживаемые файлы
git commit -m 'Name of the commit' - закоммитить файлы в обход текстового редактора
git log - посмотреть историю коммитов
git log --pretty=short - сделать историю коммитов в сжатом виде
git log --pretty=medium - сделать историю коммитов в более развернутом виде
git log --pretty=full - сделать историю коммитов в полном виде
git log --pretty=fuller - сделать историю коммитов в очень развернутом виде
git log --pretty=oneline - сделать историю коммитов в одну строку
git log --pretty=format:"%h -> %an : %s" - сделать историю коммитов, где будут показаны сокращенный хеш коммита, имя авторов и содержание коммитов
git diff - показывает различие между версиями измененных файлов
git diff --staged - показывает различие между версиями подготовленных файлов
fit reflog - выводит все наши действия
git config --list - посмотреть настройки
git remote add origin https://github.com.... - связаться с удаленным репозиторием
git remote -v - посмотреть на удаленные репозитории
git push origin main - отправить (запушить) в репозиторий
git clone - забирает чужой репозиторий и создает новый
gh repo clone - склонировать чужой репозиторий
gh repo create - создать новый репозиторий
